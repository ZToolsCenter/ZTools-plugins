# 肉鸽射击 (Rogue Shooter)

一款快节奏的肉鸽射击游戏，升级武器和技能，在无尽的敌人浪潮中生存！

## 🎮 游戏玩法

### 基本操作
- **移动**: W/A/S/D 键控制角色移动
- **瞄准**: 鼠标移动控制射击方向
- **射击**: 自动射击，无需按键
- **暂停**: 空格键暂停/继续游戏

### 游戏目标
- 在无尽的敌人浪潮中生存尽可能长的时间
- 击杀敌人获得经验值，升级解锁新的能力和武器
- 收集经验球来升级角色等级
- 每次升级可以从3个随机能力中选择一个

## 🔫 升级系统

### 武器升级
- **分裂射击**: 每次射击额外发射子弹
- **大口径**: 提高子弹伤害
- **极速扳机**: 提高射击速度
- **穿甲弹**: 子弹可以穿透敌人
- **爆破弹头**: 子弹击中产生爆炸，造成范围伤害
- **追踪导弹**: 子弹自动追踪最近的敌人
- **橡胶子弹**: 子弹撞墙后会反弹
- **背后防线**: 射击时同时向背后发射子弹

### 属性升级
- **轻盈步伐**: 提高移动速度
- **高速弹道**: 提高子弹飞行速度
- **急救包**: 恢复生命值并增加生命上限

## 🎯 游戏特色

- **快节奏射击**: 流畅的射击体验和移动操作
- **丰富的升级系统**: 10+种不同的升级选项
- **无尽模式**: 敌人难度会随时间递增
- **视觉特效**: 爆炸、粒子效果和光环特效
- **统计系统**: 记录击杀数、生存时间和等级

## 💡 游戏技巧

1. **保持移动**: 不要停留在原地，持续移动避免被包围
2. **优先选择**: 根据当前情况选择合适的升级
3. **收集经验**: 及时收集经验球来快速升级
4. **利用爆炸**: 爆破弹头可以有效清理密集的敌人
5. **背后防御**: 背后防线可以防止被偷袭

## 🔧 技术说明

- 基于 HTML5 Canvas 开发
- 支持 Windows、macOS、Linux 平台
- 需要 uTools 环境运行
- 游戏数据自动保存

- `空格键`: 暂停/继续游戏
- `W/A/S/D`: 角色移动
- `鼠标移动`: 瞄准方向
