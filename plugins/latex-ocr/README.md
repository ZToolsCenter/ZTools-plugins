# latex 编辑器&公式识别

编辑器基于[LaTeXLive](https://www.latexlive.com/home)的[1.6.4开源代码](https://github.com/QianJianTech/LaTeXLive), 修改OCR api为[SimpleTex 公式识别标准版](https://simpletex.cn/api_doc)

在编辑器设置中填入**授权令牌**即可使用OCR功能, 授权令牌可在[SimpleTex API开放平台](https://simpletex.cn/user/center)中获取

api定价仅供参考，以官方为准

| api接口          | 免费额度   |
| ---------------- | ---------- |
| 轻量公式识别模型 | 每日2000次 |
| 标准公式识别模型 | 每日500次  |

未填写api令牌不影响公式编辑器功能
